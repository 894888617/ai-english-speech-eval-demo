## ASR 是否准确

- [ ] `Hello.` 返回 `Hello.` 或按 Mock 规则返回 `The.`，可用于演示错读。
- [ ] 多词句子 ASR 文本能展示在右侧结果区。
- [ ] ASR 文本与目标句不一致时，演示人员能解释这是 Mock 错读效果。

## 错词是否能识别

- [ ] 单词级结果有数据。
- [ ] 低于 70 分的词显示红色。
- [ ] 70–85 分的词显示黄色。
- [ ] 中文建议中包含低分词重点练习提示。

## 儿童录音是否可用

- [ ] 浏览器能申请麦克风权限。
- [ ] 儿童朗读后可以停止录音。
- [ ] 待评测音频播放器能回放儿童录音。
- [ ] 录音提交后能返回评分和建议。
- [ ] 环境噪声较大时仍能完成 Mock 评测流程。

## 是否建议进入正式小程序开发

- [ ] 业务方认可后续接入真实语音评测 API 的方向。
- [ ] 已明确小程序端录音、上传、授权和儿童隐私合规方案。
